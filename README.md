# 🌿 Smart Leaf Disease Detection System

## Project Overview

This project detects plant leaf diseases using Deep Learning (MobileNetV2) and provides disease solutions and fertilizer recommendations.

## Features

* Disease Detection from Leaf Images
* 15 Disease Classes
* MobileNetV2 Deep Learning Model
* Streamlit Web Application
* English, Hindi and Marathi Language Support
* Fertilizer Recommendation System
* Power BI Dashboard

## Technologies Used

* Python
* TensorFlow
* MobileNetV2
* Streamlit
* Power BI
* PlantVillage Dataset

## Dataset

PlantVillage Dataset containing 15 disease classes of Tomato, Potato and Pepper leaves.

## Model Performance

* Accuracy: 89.06%
* Total Images: 4122
* Correct Predictions: 3671
